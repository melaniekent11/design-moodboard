# Design Moodboard

## Project Description
Design Moodboard is a web application that allows UX/UI designers to save and organize design inspiration for different projects. Users can create inspiration entries, categorize them by project and design type, and manage them through editing, deleting, and filtering tools. The goal is to help designers keep creative references organized in one place for future design work.

## Planned Features
1. Add inspiration entries with a title, project name, category, source link, image URL, and notes
2. View saved inspirations in a card-based layout
3. Edit existing inspiration entries
4. Delete inspiration entries
5. Filter inspirations by project name or category
